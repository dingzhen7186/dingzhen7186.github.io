关于python3和python2:

其中python开发者说过python2版本会在近几年淘汰掉,所以我们现在学习python首当其冲可定是安
装python3。

python3环境变量配置：

  1. python安装包路径添加到path变量中
  2. python安装目录下的Scripts路径添加到path中(不配这个pip3工具用不了)

关于pip:

pip是一个现代的，通用的python包管理工具。提供了对python包的查找，下载，安装，卸载的功能。
其中pip一般指python2的安装工具，而pip3则是python3的工具
在Windows系统下，如果你安装的是python2，那么你的工具就是pip，如果安装的是python3，那么你
既有pip，又有pip3工具。使用pip安装库的话会安装到python2.X\site-packages里面，使用pip3的话
会安装在python3.X\site-packages，如果你的电脑只安装了python3，效果其实是一样的，但还是建
议使用python3的pip3工具安装。
